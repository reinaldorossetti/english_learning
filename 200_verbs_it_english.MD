## Study Tips (Dicas de Estudo)

- Daily routine: Study 20–30 minutes every day. | Rotina diária: Estude 20–30 minutos por dia.
- Learn by sections: Focus on 20 verbs per section. | Estude por seções: Foque em 20 verbos por seção.
- Spaced repetition: Review on days 1, 2, 5, 10, 20. | Repetição espaçada: Reveja nos dias 1, 2, 5, 10, 20.
- Pronunciation: Say verbs aloud using the phonetic guide. | Pronúncia: Fale os verbos em voz alta usando a guia fonética.
- Make sentences: Write 1–2 sentences per verb using tech context. | Frases: Escreva 1–2 frases por verbo usando contexto de TI.
- Practice tenses: Conjugate in present, past, and future. | Tempos verbais: Conjugue no presente, passado e futuro.
- Flashcards: Front = ID + Verb; Back = translation, tenses, IT context. | Cartões: Frente = ID + Verbo; Verso = tradução, tempos, contexto de TI.
- Active recall: Test yourself without looking at the table. | Recordação ativa: Teste-se sem olhar a tabela.
- Context & emotion: Link verbs to real IT situations at work. | Contexto & emoção: Relacione verbos a situações reais de TI no trabalho.
- Mini-dialogues: Combine 5 verbs to create quick tech dialogues. | Mini-diálogos: Combine 5 verbos para criar diálogos rápidos de tech.
- Error log: Keep a list of verbs confused in meetings/PRs. | Registro de erros: Mantenha lista de verbos confundidos em reuniões/PRs.
- Rotate skills: Listen, speak, read, write each session. | Gire habilidades: Ouvir, falar, ler, escrever em cada sessão.

**Priya:** Good morning team! We need to **plan** our sprint. Carlos, did you **estimate** the user stories?

**Carlos:** Yes, I **estimated** them yesterday. The authentication feature is 8 points and the database migration is 5.

**Priya:** Great. Ana, can you **review** the test cases before we **start** coding?
